# RESUME

此資料夾存放履歷文件，版本控管。

## 檔案規劃

| 檔案 | 說明 |
|---|---|
| `resume-zh.md` | 中文履歷（Markdown 源檔） |
| `resume-en.md` | 英文履歷（Markdown 源檔） |
| `resume-zh.pdf` | 中文履歷輸出版（不 commit 草稿，只 commit 正式版） |
| `resume-en.pdf` | 英文履歷輸出版 |

## 更新原則

- 每完成一個重要項目或技術里程碑，同步更新履歷
- PDF 用 Markdown → PDF 工具輸出（如 pandoc 或 Typora）
- 版本號用日期標記（例如 `resume-en-20260402.pdf`）
